# 画像配置レイアウトアプリ

L判・2L判の用紙サイズに合わせて、複数の画像を自由に配置・トリミングできるブラウザベースのアプリケーションです。
印刷用に高い解像度（300ppi）で画像を出力できます。

## 特徴
- **高解像度出力**: 印刷に適した300ppiでキャンバスを作成・保存します。
- **用紙サイズ対応**: L判 (89x127mm) と 2L判 (127x178mm)、縦横の切り替えに対応。
- **ドラッグ＆ドロップ**: PC画面へのドラッグで簡単に画像をアップロード可能。
- **高機能トリミング**:
    - mm単位でのサイズ指定（即時反映）
    - 画像の移動・拡大縮小（パン操作）
    - 自由な比率での切り抜き
- **精密なレイアウト**: キャンバス上の移動は1mm単位でスナップするため、位置合わせが簡単です。

## 使い方

1. **アプリを開く**
   フォルダ内の `index.html` をブラウザ（Chrome推奨）で開いてください。

2. **用紙設定**
   画面左上の設定で「用紙サイズ」と「向き」を選択します。

3. **画像のアップロード**
   - 「画像をアップロード」ボタンを押してファイルを選択するか、
   - 画面内に画像ファイルを直接ドラッグ＆ドロップしてください（最大4枚まで）。

4. **画像の編集（トリミング）**
   登録された画像の「編集」ボタンを押すと、編集画面が開きます。
   - **モード切替**: 「自由/比率」または「mm指定」を選べます。
   - **mm指定**: 幅と高さを入力すると、そのサイズ比率で枠が固定されます。
   - **移動モード**: 画像を拡大している場合、ツールバーの「✋ 移動」を選ぶと表示範囲をドラッグで動かせます。
   - **削除**: 「この画像を削除」ボタンで、キャンバスから取り除けます。
   - **決定**: 「決定」ボタンで編集を確定します。

5. **レイアウト調整**
   プレビュー画面上の画像はマウスでドラッグして移動できます。1mm単位でカクカクと動きます。

6. **保存**
   画面下の「PNGで保存」または「JPEGで保存」ボタンを押してください。
   ファイル名には日時が付与されダウンロードされます（例: `layout_image_20251215_153000.png`）。
